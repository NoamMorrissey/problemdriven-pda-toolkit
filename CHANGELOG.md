# Changelog

## [0.2.0](https://github.com/NoamMorrissey/problemdriven-pda-toolkit/releases/tag/v0.2.0) — Beta 0.2

Initial public release of the PDA Toolkit.

### Features

- 4 PDA agents: `pda-problem` (validates), `pda-solution` (validates), `pda-context` (validates), `pda-ai-build` (generates and builds)
- Multi-agent support: Claude Code, GitHub Copilot, Cursor
- Complete TeamTasks example with research evidence, Problem Statement, Solution Brief, and Context Specification
- Empty templates for starting your own project
- PDA Philosophy rule enforcing human-write / AI-validate separation
- Documentation: README, QUICKSTART, PROCESS, HOW-AGENTS-WORK, FAQ, METHODOLOGY, EXAMPLE
